# Bookmark Landing Page

A **responsive** and **accessible** landing page for a simple bookmark manager web app.  
This project demonstrates skills in **HTML, CSS, and vanilla JavaScript** while prioritizing accessibility and mobile-friendly design.

## Features
- Fully responsive layout for **desktop, tablet, and mobile** screens
- Accessibility-friendly design (semantic HTML, keyboard navigation, ARIA attributes)
- Clean and modern UI
- Interactive FAQ section
- Browser support: Chrome, Firefox, Opera
